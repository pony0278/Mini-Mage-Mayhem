## 1. 為什麼咬合度高(幾乎是現成的)

1. **三階段升級已存在**:場內收容升級「普通→黃色警戒→全面失控」(`applyStage`:桶更快/艙吸力更強/站更頻)
   → 攤開成三個關卡,骨架直接沿用。
2. **廠長早就在場**:開場訓話「🧑‍💼 主管:都給我好好工作!」——玩家第一秒就認識反派;最後一關打他=零成本敘事回收。
3. **「下班」結局概念現成**:B 款凍結時做過整套打卡下班結局(4c92837),概念復活成闖關結局。
4. **終極笑點**:最後一擊=把廠長塞進中央回收艙——「員工把老闆當垃圾回收掉然後下班」,
   北極星「好笑到想截圖」在此封頂。

## 3. 廠長 boss 設計

**體格**:放大 1.5~2× 的 brawler(rig 可縮放)。**開發先用放大現有模型+現有 clips 頂著;
使用者的廠長建模(assets 管線)+ PUNCH STUDIO 專屬動作好了隨時換皮——兩邊平行不互卡。**

**核心機制(全部長在現有系統上)**:
- **太重不可抓**(startCarry 對 boss 免疫)、**鉤拳打不暈**(對 boss 削穩定極少或 0)。
- **削他的穩定=技術動作**:反擊拳(擋他的大拳→窗內反打)、下壓拳、環境(爆桶/元素站/冰滑撞牆)、
  他自己衝撞撞牆(見招式)。= 前兩關學會的所有東西在 boss 戰全部用得上(課程設計)。
- **三段擊倒循環**:穩定歸零=擊倒(倒地可追打數秒)→ 起身變兇(招式加速/解鎖新招)→ ×3 後進
  **「過載狀態」**(延長暈眩+可被挑飛/風壓/環境推進艙)→ 塞進回收艙=通關(收容演出加大號罩)。

**招式(機制我做,clip 使用者後補;全部復用現有判定原語)**:
| 招 | 機制原語 | 反制 |
|---|---|---|
| 大掃拳 | 寬錐版 resolveStrike | 擋下→反擊拳(主要削穩定來源) |
| 踩地震波 | 圓形 AoE(dive 判定的 boss 版) | 跳(空中免震波=跳躍的 boss 戰意義) |
| 衝撞 | 直線 charge(冰滑鎖直線的復用) | 側跳閃 → 他撞牆=大硬直+自削穩定(懲罰窗) |
| 丟公文箱 | 丟桶管線復用(拋物線) | 接不到就閃;落點畫圈(現有 marks) |

## 4. 結局+重玩性

- **結局**:廠長收容演出 → 打卡下班(B 款下班結局的概念復活:打卡機+下班橫幅+走出大門)→
  **全程總評事故報告**(每關報告+總評;等級 S~E 現成)。
- **重玩鉤子**:總評等級(刷 S)+ 通關時間(速通)+ 被回收次數——一次性劇情的缺點用分數系統補掉,
  portal 玩家有理由重打。

## 5. 實作計畫(每步照慣例:tests+文件+build tag+推兩分支)

1. **闖關① 殼**:campaign 狀態機(關卡進度/過關→轉場→下一關/敗北重試/localStorage)、關卡標題卡、
   過關結算(每關報告)。現有 restartMatch/resetRound 復用。
2. **闖關② 關1/關2**:對手 AI 分檔調參(關1 慢/關2 兇)、場地危險物按關卡開關(applyStage 對映)。
3. **闖關③ boss**:大隻佬 actor(縮放)+ 不可抓/鉤拳免疫 + 四招機制 + 三段循環 + 過載收容。
4. **闖關④ 結局**:打卡下班演出 + 總評報告 + 加班模式解鎖。
5. **闖關⑤ 換皮**:使用者的廠長模型/clips 進場(extractor→punch-studio→遊戲管線)。
