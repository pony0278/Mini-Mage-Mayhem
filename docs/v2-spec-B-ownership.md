# v2 規格 B — 鋪設→累積→引爆（ownership / 囤積感）【測試計畫】

> 對應：v2《環境處決 PvP》§0.5 B（很痛・**該測非該加**）、§8（鋪設期/引爆期）、§9-3 + 紅燈三（卡門地獄風險）、§12。
> 一句話：**ownership 從「即時動詞留下的持久痕跡」長出來，不要做「布置模式」；且必須靠 `lastHitBy` 歸因讓預埋被看到。護欄沒守住就砍。**
> 狀態：**測試計畫**（B 是「該測非該加」，本文不是實作決定，是一次 A/B 測試的設計）。前置：規格 A（好笑）先過。

---

## 1. B 買到什麼
- **ownership / IKEA 效應**：「我 30 秒前埋好局、等你踩」的延遲 payoff，是「玩家當作者」軸的**放大器**。即點即爽轉瞬即逝；預埋成功變成一個**可炫耀、可分享的故事**。
- 天然製造 §5 的**慌張視窗**：陷阱已在那，受害者踩進去到引爆之間 = 「看得到死、逃不掉」那幾秒戲。

## 2. 張力（紅燈三，必須正視）
純布置 → **卡門地獄**：龜縮、節奏拖沓、移動煩，失去 io 的即點即爽。而原型「即噴即點」**已驗證會笑**。

> 鐵則：**B 不是用「布置」取代「即時」，而是在即時之上，讓動作留下可累積、可後觸發的痕跡。**

## 3. 解法：ownership 從「持久狀態」長出來，不做「布置模式」
- **不新增**慢吞吞的「擺放陷阱」動詞。
- 讓**現有即時動詞留下持久場上狀態**：現在噴油（即時、爽）→ 油留著、可再噴成一片（累積）→ 有人踩 / 有人點火（引爆）。
- 延遲**自然發生在「我鋪了」和「你踩到/我點燃」之間**，不需要回合、不需要模式。**即點即爽完全保留**，ownership 是附加層。
- 與 [v2-spec-elements.md](v2-spec-elements.md) 相扣：空間動詞本來就是地板 zone → **B 多半只是調「持久 + 累積」＋ 加炸藥桶，不是新系統**（便宜）。

## 4. 要測的旋鈕（起始值，A/B 用）

| 元素/物件 | 即時層（保留，always-on） | 累積 / ownership 層（要測的變數）| 起始值 |
|---|---|---|---|
| 油 | 噴一灘、可立刻點 | 持久 + 相鄰噴**連成一片**；點火沿油快速燒成火海 | 持久 **10s**；連片 **on**；火沿油蔓延速度 ~中 |
| 毒區 | 即放即傷 | 持久、可疊濃；慢性扣血=慌張計時器 | 持久 **8s**；可疊 **2 層** |
| 電水 | 即電即麻 | 預先充電一條水路（持續）逼人繞/自電 | 充電持續 **6s** |
| 冰面 | 即凍 | 預凍逃生路 → 追兵滑進空洞 | 冰面持久 **10s** |
| 炸藥桶 | —（純物件）| **可搬/可推、預先囤路口**；遇火/爆炸連環炸 | 場上常駐 N 顆；爆半徑 ~中；連鎖 **on**（你朋友自己發明的玩法＝強訊號）|

「累積/囤積」= (a) zone 持久且能連片變大，(b) 桶可搬來囤。「延遲」= 鋪好→引爆之間。

## 5. ownership 必須「被看到」（不可省）
預埋成功若沒被歸功，IKEA 效應就沒兌現。**B 必須接規格 A 的 `lastHitBy` + 最大災難回顧（C）**：
- 「**你 30 秒前鋪的油燒死了 X**」/「X 踩進你預埋的電水自己麻死」= 炫耀句 + 截圖標題。
- 自踩自爆 → 歸因 self →「X 自己掉進自己的局」（自作自受，更好笑）。

## 6. 防卡門地獄的護欄（守不住就砍累積層）
- **即時層永遠在**，fun 不鎖在預埋後面。
- **焦點物（搶結晶）逼人離開陷阱出來互搶** → 不能龜。
- **縮圈 + Boss 趕人** 維持 tempo，把人擠進別人布好的局。
- **移動要快、爽**——移動一變煩，B 就失敗。

## 7. 測試計畫（這才是 B 的交付）
1. 同一張瓶頸地圖，出兩版：
   - **(a) 即時為主**：油 ~3s 就消、不連片、無常駐桶。
   - **(b) 即時 + 累積**：第 4 節的起始值（油 10s 連片、可囤桶…）。
2. 找人玩，看的不是輸贏，而是 **玩家會不會「自發」開始預埋**（像朋友自己發明搬桶丟火）。
3. **正指標**：自發預埋次數、「那是我鋪的！」這類發言、笑出聲、主動再玩。
4. **反指標（= kill-criteria）**：有人開始龜、抱怨移動煩、節奏停滯、引爆等太久。
5. **若反指標贏** → 砍累積層，回到「即時為主、持久當佐料」（呼應紅燈三 + §9-2「重心可能要反過來」）。

## 8. 紀律
- B 在 **A（好笑）gate 之後**；A 沒讓 3–4 人笑出聲前，不蓋陷阱系統。
- B 本身就是一次 A/B 測試，**不是默認要加**。
- 不把它做成 build 深度（每張元素卡仍只問「能對別人空間做什麼壞事」）。
