# Mini Mage Mayhem — 遊戲介紹總覽

> 一句話：**單人、網頁、3D 俯視角的「元素魔法 Roguelike」**——用 4＋1 種元素自由融合出你的法術，把混亂的元素反應變成武器，在 5 波敵潮後挑戰元素 Boss。
>
> 線上試玩：<https://pony0278.github.io/Mini-Mage-Mayhem/>　|　平台：瀏覽器（桌機 + 手機觸控）

---

## 1. 核心賣點（Elevator Pitch）

- **元素融合系統**：5 種元素（火・冰・雷・毒・土）兩兩融合出 **10 種獨特法術**，加上單元素共 **16 種主法術**。
- **元素反應即武器**：火點燃草地成火海、雷在水池導電、火引爆毒霧連環炸、土砸碎牆改變路線——**戰場本身會反應**。
- **三條近戰流派**：不想當遠程法師？把主攻換成 **土拳 / 雷掌 / 風掌**，各有 **3 段連段普攻 ＋ 蓄力重擊**，玩起來像三款不同的近戰遊戲。
- **畢業大絕**：把 build 投資到位，解鎖 **10 種被動大招**（流星降臨、絕對零度、磁暴奇點……）。
- **高風險高回報**：強力魔法常常**也會傷到自己**（踩水自電、過載自傷、大爆波及），玩的是「掌控混亂」。
- **短局 Roguelike**：一局 5 分鐘上下，每波三選一升級，build 千變萬化。

---

## 2. 遊戲流程（一局的節奏）

```
標題畫面
  → 隨機抽一張「場地」（4 選 1，各有不同地形與反應傾向）
  → 第 1～5 波敵潮
       每波開始：隨機觸發一個「波次事件」（長草 / 滲水 / 暴走 / 過載…）
       每波結束：3 選 1 升級（注入元素 / 修飾 / 衝刺 / 副攻 / 轉職 / 精通 / 畢業大絕）
  → 第 5 波後：Boss「元素哥布林法師」登場（元素連招 + 第二階段）
  → 擊敗 Boss → 勝利結算　／　血量歸零 → 死亡結算（含死因 + 最大災難回顧）
```

Roguelike 單局制：每局重新組 build，場地、波次事件、升級牌都隨機。

| 🔥 火 | 火球 | 點燃草地、引爆毒霧、留火焰地板 |

## 5. 近戰流派（把主攻換成肉搏）

每個流派 = **3 段連段普攻（連點循環）＋ 長按蓄力重擊**，並可重選升星（★1→★3）。

| 流派 | 3 段普攻「連段」 | 蓄力重擊（長按）| 升星與招牌 |
|---|---|---|---|
| **土拳** 崩石三連拳 | 左重拳 → 右勾拳（往瞄準方向打飛）→ 雙拳砸地（扇形震波破牆砸箱）| **裂地上勾拳**：擊飛 + 裂地破牆、打斷衝鋒 | ★2 震波、★3 週期地震；撞牆追加、破牆碎石 |
| **雷掌** 閃雷三段斬 | 直線手刀（上雷印）→ 橫斬（雷鏈連敵）→ 雷步穿身（穿越到背後＋引爆雷印）| **雷閃穿刺**：範圍內連續閃現穿刺多名，再閃回原點 + 終點電爆 | 雷印、穿身引爆、水池導電、★3 雷神週期放電 |
| **風掌** 流風三連掌 | 推掌 → 旋掌（把敵人往側邊帶）→ 氣爆掌（錐形風壓 + 吹散火毒蒸氣 + 推物 + 撞牆）| **真空掌**：蓄力吸引聚怪，再一掌徑向氣爆吹飛（Boss 只硬直）| E 撿取敵人/木箱「齊射」丟出；★ 一次撿更多 |

近戰共通：點擊普攻會**自動撲向瞄準錐內最近敵人**；命中有頓幀與火花回饋。

## 7. 升級系統（每波三選一）

- **注入元素**（火/冰/雷/毒/土）：改造或融合你的主法術（近戰時改為強化該元素近戰效果）。
- **修飾**：分裂、命中爆炸、元素留痕、快速施法、巨大化、災難回血、元素蔓延、毒爆擴大、導電麻痺、大冰面、寒冰碎裂、危險魔力（高傷高自傷）。
- **衝刺**：疾風步（冷卻）、衝刺強化、疾風連步（連續衝刺）。
- **副攻解鎖**：冰牆 / 土牆 / 潑油 / 黑洞。
- **肉搏轉職**：土拳 / 雷掌 / 風掌（重選＝升星）。
- **通用**：強健體魄（血量）、迅捷（移速）、回春（回血）。
- **元素精通**：重複投資同元素 → 該元素法術逐級增強（雷鏈跳數、火蔓延等）。
- **畢業大絕**：build 投資到位才會出現的被動大招（見下）。

## 8. 畢業大絕（Capstones，10 種）

把對應兩元素投資到位即可解鎖一個一局一次的被動大招：

| 元素組合 | 大絕 | 效果 |
|---|---|---|
| 火 + 土 | 流星降臨 | 持續天降流星，落點預警後爆炸並留岩漿池 |
| 火 + 毒 | 瘟疫核爆 | 你佈下的毒霧會週期性自動連環引爆 |
| 土 + 雷 | 磁暴奇點 | 週期生成磁力奇點吸敵成團，塌縮時雷鏈貫穿 |
| 冰 + 毒 | 凍毒領域 | 身周凍毒光環，範圍內冰緩 + 中毒 |
| 火 + 雷 | 電漿風暴 | 電漿球在場上自走，週期爆裂並雷鏈導電 |
| 土 + 冰 | 冰川崩落 | 敵群外圍升冰牆關籠，隨即碎裂成大冰爆 |
| 火 + 冰 | 沸騰領域 | 全場蒸氣風暴，所有敵人持續減速並灼燒 |
| 雷 + 冰 | 絕對零度 | 週期釋放冰凍新星，凍結並電擊全場 |
| 雷 + 毒 | 劇毒電網 | 身周帶電毒場，中毒 + 週期麻痺 |
| 土 + 毒 | 大地崩毀 | 腳下毒沼擴張，陷足中毒並迸碎石擊退/破牆 |

## 9. 敵人與 Boss

| 敵人 | 特性 |
|---|---|
| 史萊姆 | 基礎追擊雜兵 |
| 毒蟲 | 速度快，**死亡噴出毒霧** |
| 火焰小鬼 | **遠程火彈**攻擊 |
| 盾甲衝鋒怪（菁英）| **正面格擋減傷、背後弱點加傷**、可被暈眩打斷衝鋒 |
| **Boss：元素哥布林法師** | 用毒/火/雷/水的**預警圈連招**改變戰場，第二階段更兇；先讀地上的預警圈再走位 |

敵潮配置（範例）：第 1 波史萊姆 → 逐波加入毒蟲、衝鋒怪、火焰小鬼 → 第 5 波後 Boss。

## 10. 場地（4 種，各有反應傾向）

| 場地 | 特色 |
|---|---|
| 失控溫室 | 草地多，火焰快速形成大片火海 |
| 煉金毒液地窖 | 毒霧與毒蟲多，火元素容易連鎖爆炸 |
| 淹水魔法聖壇 | 水池多，雷鏈很強；冰可把水面凍成滑冰場 |
| 崩壞訓練場 | 薄牆多，爆炸與衝鋒可打穿、改變路線 |

### 波次事件（每波隨機）
草地蔓延 / 毒霧補給 / 地板滲水 / 寒霜裂縫 / 暴走小怪（更多更快）/ 魔力過載（傷害↑自傷↑）/ 菁英巡邏（多一隻衝鋒怪）。

## 11. 元素反應與環境互動（系統亮點）

- **火 × 草地** → 點燃成大片火海　**火 × 毒霧** → 連環引爆
- **雷 × 水池** → 整池導電（站在裡面會電到自己）　**冰 × 水池** → 結成滑冰場（敵我都會滑）
- **冰 × 火** → 冰牆/冰面融化成蒸氣
- **爆炸 / 土系** → 打碎薄牆、改變戰場路線；砸木箱迸碎片
- **風掌** → 把火/毒/蒸氣雲、木箱、敵人往前吹（可把敵人吹進火場、或撞牆）
- **帶電木箱** → 雷掌充能木箱成移動電牆
- **高風險設計** → 危險魔力/魔力過載提高傷害也提高自傷；大爆炸與電場會波及自己

## 12. 手感與呈現

- **3D 體素風 + 固定 45° 跟隨攝影機**（Three.js）。
- **打擊感打磨**：頓幀（hitstop）、命中火花、傷害縮放閃光、螢幕震動節流（小事不狂震、大事才重震）。
- **程序化音效**（WebAudio，無音檔）。
- **觸控完整支援**：雙動態搖桿 + 按鈕。
- **雙語就緒**：英文預設 + 中文字串（gettext 風格）。

## 13. 技術規格（給開發/上架用）

- 純前端、**無框架、無打包**：build-free ES modules（`js/`）＋ vendored Three.js。
- 模擬與渲染分離：`sim.js`（純邏輯、可無頭執行）↔ `render.js`（3D + HUD）。
- 部署：GitHub Pages（main 分支根目錄）；已備 CrazyGames SDK 接入（廣告/生命週期，無 SDK 時優雅降級）。
- 規劃中：上架 CrazyGames 收留存數據 → 評估是否往「即時多人大逃殺（BR）」演化（sim 已為無頭化鋪路）。
