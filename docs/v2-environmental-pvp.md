## 0.5 目標再聚焦（prototype 後，這節最該先讀）

### 已發生的驗證（別忽略它的份量）
- 把核心做成最醜熱座雙人原型（噴油 / 噴火 / 搬炸藥桶 / 搶獎盃回安全區），給一個真人玩。
- 結果：她不只覺得好玩，**主動加碼設計**（雙方都能噴火噴油、地圖放炸藥桶可搬、丟到火上、來不及的人被炸到「凸眼」超好笑）。
- 意義：一個素人玩家憑直覺，重新發明了本文件的好幾條核心（玩家當作者、延遲引爆、看好戲軸、蠢死法、手殘也好笑）。**這是核心鉤子第一次被現實阻力戳到——而它沒垮，反而被印證。**
- 原型已做對的事（要守住、放大時別弄丟）：真的做了減法（沒有 16 法術 / 近戰連段 / build / 大絕）；雙方對稱＝「環境亂鬥」而非「我陰你」（對陌生人局友善）；爆炸不分敵我（自作自受寫進規則）；Boss 平時睡、搶盃才醒、只追抱盃者（Boss 已降級成催化劑，不是血條）；桶炸桶、火燒油的連環鏈已成立。

### 北極星（再聚焦版）
> **一場 2–4 人的混亂奪旗亂鬥：玩家用元素『動詞』改變地形、互相把對方炸飛進火海或空洞（玩家當作者、有受害者、自作自受）；死得越蠢越好笑、越能一鍵分享。Boss 與縮圈只負責把人擠向焦點。先用熱座驗好笑 → 再驗同步 → 最後才提交完整棧。**

### 兩條方向的精確化（修正容易走歪的框架）
1. **「卡片獲得元素」≠ build 變強。** 卡片給的是「改變對方所處空間的新動詞」，不是更高 DPS。火＝點燃你腳下的油；冰＝把你的退路凍成滑冰場滑進空洞。每張卡問：它讓我能對別人的空間做什麼新的壞事？（用守則第 4 節那把尺刷每一張卡。）
2. **「奪旗」對照作不是 Fall Guys。** Fall Guys 是障礙賽淘汰、玩家幾乎不互動、混亂來自系統機關——與本作相反。本作混亂來自玩家互相用環境陷害。獎盃的功能是「製造一個所有人都想搶、因此聚在一起互炸的焦點」，不是各跑各的。對照作應是「奪旗 + 玩家用環境互害」（近 Smash Bros 搶東西亂鬥）。

### 漏掉清單（已討論認同、但還沒落進原型；按漏掉會痛的程度排序）
- **A（最痛・下一個動作）受害時刻的畫面糖果 / 蠢死法。** 你親口承認缺、朋友親口點名要、原型還沒做。被炸飛出界要凸眼、慘叫、滑稽飛出去。沒有可截圖的醜態＝沒有 TikTok。這是分享性核心，不是裝飾。**第一優先，比再加一個元素重要。**
- **B（很痛・該測非該加）「鋪設→累積→引爆」的延遲與囤積感。** 原型現在即噴即點、即搬即炸，少了「我預先埋好局等你踩」的 ownership（IKEA 效應）。測一版：油可累積成片、桶可預先囤路口。注意與「即點即爽」有張力（紅燈三），所以是「該測」不是「直接加」。
- **C（會痛・增長引擎）最大災難回顧 / 一鍵分享。** v1、v2 都寫過，原型還沒有。把「好笑的一局」轉成可發佈影片的轉換器。投報率可能比再加元素高，因為它直接把爆笑瞬間做成素材塞給玩家。
- **D（未來・有意識地漏）同步 / netcode。** 熱座雙人對「驗好笑」是正確選擇，現在就是該漏。鐵律仍是 fun → netcode → stack。等多人都笑了，它才變下一個 Probe。
- **E（中後期・別現在做）給創作者炫耀的舞台。** 讓玩家秀並解說自己的鬼點子設計，吸引自帶粉絲的創作者。現在做會分散驗證核心的力氣；記著別忘當初為何重要。
- **F（平台・等資料）Poki vs Steam 分叉 + 「陌生人無語音好不好笑」最脆假設。** 「對稱亂鬥」已部分緩解 rage quit 風險，但陌生人到底笑不笑仍只有真人多人測試能答，且直接決定平台。別現在鎖死。

### 下一個動作（別回到純討論）
做 A（空洞墜落 + 誇張炸飛/凸眼死法動畫——原型的 CFG / goDown / knockEntity 零件都在），然後**再找 2–3 個人玩**，看「凸眼飛出界掉下去」是否真讓人笑出聲。把原型變成「有人笑」的數據。

## 0. 北極星（v2 版）

> **保留單機核心當骨架（身體），換掉終局與對抗層（頭）：終局是一個「玩家用環境把彼此送死、但動手的是怪／場地／Boss」的短局混亂逃脫。**

## 1. A / B 路線決定

| | Path A（離開） | Path B（選擇） |
|---|---|---|
| 形態 | 單機 roguelike + 打 Boss | 多人、環境處決 PvP |
| 重玩引擎 | build 多樣性（我 vs 系統） | 玩家互動（我 vs 你，透過場地） |
| 增長 | 買量 UA + 變現深度 | 病毒影片 + 口碑 |
| 變現 | 深養成 / 內購 | Poki 免費 + 廣告（暫定，見脆弱點一） |
| 對照作 | 弓箭傳說 | Smash Karts / The Headliners |

**決定：傾向 B。** 但 B 不是「把 A 簡化」，而是**把複雜度從一條軸搬到另一條軸**——詞彙變少、博弈變深（見第 3 節）。

⚠️ 警告：A 的「build 失控爽感」是你熟悉、做起來爽的領域，會持續把你拉回 A。每次想加 build 深度時，回來看這格。

## 2. 核心模型：共存 + 競爭，系統當執行者

借自 The Headliners 的關鍵洞察：**怪物／環境當「處決者」，玩家當「設計師」。**

- 不是純 PvP 互殺 → 太傷人、rage quit。
- 不是純 co-op 合作 → 混亂變成系統的、需不斷餵內容、膩得快。
- 是「**一起求生，但搶有限資源**」：玩家是共同求生者，也是競爭者。惡意包在共存的糖衣裡。

**為什麼這個中間解最優（理論上）：**
- 保留「看好戲」軸（有真人受害者會慌）→ 最高分享性。
- 保留「玩家是混亂作者」（IKEA 效應）→ 不用一直餵內容。
- 因為「不是我親手殺的、是怪/場地殺的」→ 責任分散，避開最硬的 rage quit。

## 3. 複雜度搬軸：誰塌縮、誰加深

B 之下，總複雜度可以不變，但分配整個翻過來。

**塌縮（砍或大幅簡化）— 全是「我 vs 系統」的縱深：**
- **10 種畢業大絕** → B 之下最不合身，幾乎全砍。理由：被動自動觸發（玩家沒在操作）、不可反制、對手看不到、連續 AoE 最難同步。低技 + 不可讀 + 難同步。
- **三套近戰流派的「格鬥連段系統」** → 砍精度面（見第 6 節，但保留環境面）。
- **16 種法術** → 壓到對手能秒讀的小詞彙表（約 4–6 種，每種威脅一眼可辨）。可讀性是 PvP 硬天花板。

**加深（從「一行字」做成主菜）— 全是「我 vs 你，透過場地」的縱深：**
- **冰牆 / 土牆 / 潑油 / 黑洞**（空間否定工具）→ 從邊角變核心動詞。
- **延遲引爆陷阱**（油、電水、毒霧、冰面：先布置、後觸發）。
- **環境反應與瓶頸地圖**。

## 4. 三層傷害來源（刻意分層）

| 層級 | 角色 | 設計意圖 |
|---|---|---|
| **環境**（火/毒/電/冰/Boss/崩塌）| **主力擊殺** | 你想要玩家主要死於這個 |
| **法術控場**（吹/推/封/緩）| 輔助 | 把對手**送進**環境，不直接清血條 |
| **法術直擊**（連招打到對方）| 降級保留 | 還會扣血，但刻意不足以當主要擊殺手段，逼玩家用環境 |

**檢查每個攻擊動詞：** 它是在「扣對方血」還是「改變對方所處的空間」？前者降級，後者放大。

---

## 5. 血條：保留，且當「慌張計時器」

- **血條不取消。** 環境處決要成立，前提就是玩家會受傷、會死。
- **做成「環境持續扣血」，不要「碰到即死」。** 那條慢慢被環境掏空的血，就是受害者慌張、旁觀者看戲的計時器。
- 秒殺 = 沒有慌張視窗 = 沒有可截圖的掙扎 = 沒有戲。
- 環境處決要刻意做得**夠慢、夠有預兆**，讓受害者有幾秒「知道死定了但逃不掉」。

## 6. 近戰系統改造：砍格鬥面，放大環境面

用一把尺刷三套近戰：**「位移/環境動詞」留並放大，「純傷害連段」砍。**

**該砍（全系統共通）：** 3 段連段循環、蓄力重擊的精度打磨、升星 ★1→★3、逐幀 hitstop 追求。理由：餵最弱的「單純爽」軸 + netcode 最痛。

**該留並放大：**

| 流派 | 砍（純傷害/精度） | 留並放大（位移/環境）| 簡化後定位 |
|---|---|---|---|
| **土拳** | 三段連段、升星、頓幀手感 | **打飛**（轟進火海/電水/橋下）、**破牆裂地**（即時重塑瓶頸）、打斷衝鋒 | 收斂成「位移拳 + 破壞拳」兩個動詞；近距離版空間否定工具 |
| **風掌** | — | 吹散火/毒/蒸氣、推物、把敵人吹撞牆/吹進火場 | **天生最合身**，幾乎全是環境動詞，可能比土拳還核心 |
| **雷掌** | 穿身引爆、連續閃現穿刺（純連段） | 水池導電、充能木箱成移動電牆 | 留環境面、砍連段面 |

**土拳具體簡化方向：** 不要三段+蓄力+升星。收成兩招——一個**位移拳**（把對手往瞄準方向猛擊飛，送進環境）、一個**破壞拳**（砸牆/砸地，重塑瓶頸）。好讀、好同步、服務環境處決。

## 7. 地圖與場地：瓶頸拓樸

- **地圖從「開闊競技場」改成「有可被封鎖的瓶頸」**：窄橋、單一出入口的房間、橋、隘口、會滲水的迴廊。
- **前置條件原則：** 你的差異化機制（空間否定）需要**可被否定的空間**。沒有瓶頸，封門/環境處決就不存在。地圖拓樸是機制的前置條件，不是背景。
- **縮圈** = 瓶頸產生器：把開闊地慢慢擠成走廊，自然生出可被封死的隘口（不是 PUBG 的「逼大家靠近」）。
- **Boss（元素哥布林法師）降級** = 移動的危險區 / 瓶頸製造器：把玩家趕進窄道，幫玩家把獵物趕進布好的殺戮空間。**不是要被打的血條，不是主要混亂來源。**

⚠️ 避開的陷阱：「大 Boss 攪局、全員一起逃」會把混亂作者從「玩家」換成「系統」，稀釋唯一差異化，且掉進 The Headliners 那種「不斷餵新怪新圖」的內容跑步機——兩人團隊撐不住。

## 8. 一局遊玩情境（4 人局，當作願景對齊用）

**開場：** 同一張瓶頸地圖（如「淹水魔法聖壇」：祭壇 + 兩條窄橋 + 滲水迴廊）。共同目標：搶有限的「魔力結晶」並活著到出口。既共同求生，又互相競爭。

**鋪設期（0:00–1:30）：** 關鍵動詞是「布置」不是「打」。讀地圖、預判路線、把油/毒/電場/冰面**惰性地**種在別人必經之路上。等引爆。那個「我種好了等你踩」的延遲 = ownership。

**引爆期（1:30–3:30）— 看好戲在此：**
- 對手衝上你潑了油的橋 → 你點火 → 他被夾在火與橋欄間，看得到出口衝不過去，在火裡轉圈。**燒他的是橋，不是你的血條直擊。**（看好戲軸）
- 對手繞水路，但你早把水充了電 → 他踩進去自己被電硬直。（自作自受軸）
- Boss 沿迴廊巡來，把人往窄橋逼 → 幫你趕獵物進殺戮空間。
- **法術連招的角色：** 用風掌把人吹進電水、用冰牆封死退路逼他上燃燒橋。法術是「鞭子」，動手的是環境。

**崩壞收尾（3:30–5:00）：** 火海蔓延、毒霧連環引爆、薄牆炸開、縮圈擠出更多瓶頸。你抱結晶從自己冰牆封出的縫溜向出口。

**結算：** 不只名次——「最大災難回顧」+ 可分享截圖（對手在燃燒橋上轉圈）。這是病毒素材。

## 9. ⚠️ 脆弱假設（紅燈，必須盯）

以下是把方向寫得很順、但**我在替你樂觀假設**的地方。做的時候當紅燈：

1. **「責任分散讓殘忍變好笑」對陌生人成不成立。**（最脆的一塊）「對手在燒橋上轉圈很好笑」這個假設，在開語音的朋友局（The Headliners 那種）很可能成立，在 **Poki 陌生人無語音局可能整個垮掉**——你不認識那個 ID，他的慌傳不到你這裡。這直接牽動平台決定（Poki vs Steam co-op），所以平台別現在鎖死。

2. **「環境處決比法術互打好玩」是推論，不是玩家數據。** 完全可能玩家就是想痛快用法術連招對轟，覺得「布油等你踩」太繞太慢。若如此，重心要反過來：法術 PvP 當主力、環境當佐料。

3. **「延遲引爆 + 瓶頸地圖」可能變成又慢又卡的卡門地獄。** 布置玩法聽起來聰明，實作可能節奏拖沓、移動很煩，失去 io 那種「即點即爽」。

## 11. 已鎖定 / 仍開放

**已鎖定（別重複推翻）：**
- 多人架構：server-authoritative（場地狀態只由伺服器擁有，客戶端只預測自己）。封掉 desync 坑。不需要跨平台 bit-identical 決定論。
- 驗證紀律：fun → netcode → stack，配 probe/gate。
- 網頁免費玩家對 jank 零容忍（與 Steam 買斷玩家不同）。
